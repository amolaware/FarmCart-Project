# FarmCart

## Features

- User Registration & Login
- JWT Authentication
- Role Based Access
- Product Management
- Image Upload
- OTP Password Recovery
- MongoDB Atlas Integration

## Tech Stack

### Frontend
- React.js
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT
- Nodemailer

## Setup

### Backend

npm install
node Server.js

### Frontend

npm install
npm run dev
